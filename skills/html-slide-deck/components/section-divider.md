# Section Divider

## 用途

在 deck 主题切换时建立章节边界，提示下一部分要解决的问题。

## 输入参数

- `part_label`
- `section_kicker?`
- `title`
- `guide_sentence?`

## 稳定核心

- 章节页只承担转场和定向，不承载复杂内容。
- 标题应短，导语只说明下一段要解决什么。

## 常用变体

- `part-divider`: 带 PART 编号和章节标题。
- `question-divider`: 用一个问题引出下一章。
- `principle-divider`: 用一句原则或判断做章节门槛。

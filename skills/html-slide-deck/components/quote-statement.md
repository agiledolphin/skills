# Quote Statement

## 用途

表达一句核心判断、金句、原则、结论或阶段转折。适合内容很少但需要现场冲击力的页面。

## 输入参数

- `section_label`
- `title?`
- `statement`
- `support?`
- `attribution?`

## 稳定核心

- 一页只讲一句话或一个判断。
- 文字本身是主视觉，不能再叠加大量卡片、表格或说明。
- 可用于开场、章节转折、总结前的观点强化。

## 密度规则

- `statement` 最好不超过 2-3 行。
- `support` 最多 1 句，不能变成长段正文。
- 允许较大留白，但必须有强标题、强对齐或明确背景视觉承接。

## 不适合

- 多要点列表。
- 需要证明、对比、拆解或落地计划的内容。
